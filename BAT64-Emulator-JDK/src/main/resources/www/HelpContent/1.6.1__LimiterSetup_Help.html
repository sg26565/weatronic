Limiter Einstellung - passen Sie hier den Limiter an

  • Kontrollieren Sie mit dem hinterlegten Geber die Werte der Funktion in den Live Anzeigen, die Sie links oben finden.

  • Funktion, hier wählen Sie die Funktion aus, die Sie begrenzen möchten

  • Begrenzung von Oben, stellen sie einen Wert ein oder wählen Sie einen Geber, um die Funktion von oben zu begrenzen

  • Begrenzung von Unten, stellen Sie einen Wert ein oder wählen Sie einen Geber, um die Funktion von unten zu begrenzen

  • In der grafischen Ansicht werden die begrenzten Bereiche rot angezeigt
