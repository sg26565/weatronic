Konfigurieren Sie Ihren Startbildschirm nach Ihren Wünschen und erstellen Sie ebenfalls individuelle Seiten, auf denen Sie sich Telemetriedaten anzeigen lassen können.

ANMERKUNG: Sie können zwischen diesen Seiten wechseln, in dem Sie nach rechts oder links auf dem Startbildschirm mit einem Finger streichen / wischen.

  • Startbildschirm, hier können Sie Ihren Startbildschirm anpassen und zwischen verschiedenen Layouts wählen
    • Bildschirmsperre (diese Option wird demnächst erhältlich sein)
      		ANMERKUNG: Ein kleines Schlosssymbol in der obersten Informationszeile zeigt an das die Toucheingabe geperrt ist.
    • Automatische Dimmung (diese Option wird demnächst erhältlich sein)
    • Berühren Sie nacheinander die grauen Felder und wählen Sie aus, welche Daten in diesem Feld während des Fluges als Live-Daten angezeigt werden sollen
    • Sie haben auch die Möglichkeit, die komplette Darstellung der Seite Ihren Wünschen anzupassen. Berühren Sie den kleinen Button auf der linken Seite, um aus verschiedenen Vorgaben zu wählen.

  • Telemetrie - sobald Sie Telemtrie Bildschirm erstellt haben können Sie diese über die kleinen nummerierten Platzhalter auswählen und dann editieren. Hinzufügen neuer Seiten ist denkbar einfach:
    • 1st: Fügen Sie mit dem "PLUS" Button eine neue Seite hinzu, in der Telemetriewerte während des Fluges live angezeigt werden sollen
    • 2nd: Sie können sich auf einer Seite bis zu 6 verschiedene Werte anzeigen lassen – bitte wählen Sie eine der Anordnungen aus
    • 3rd: Berühren Sie erneut nacheinander die leeren Felder und wählen Sie die Telemetriedaten oder Timer, die Sie live angezeigt bekommen wollen.
