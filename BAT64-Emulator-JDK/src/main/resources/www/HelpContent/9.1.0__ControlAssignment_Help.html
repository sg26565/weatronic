Geber Zuweisung - Auswahl durch bewegen
Bewegen Sie einfach den Geber, den Sie verwenden möchten oder berühren Sie den entsprechenden Geber auf dem Bildschirm. Um keinen Geber auszuwählen tippen Sie einfach auf einen leeren Bereich auf dem Bildschirm.
Im Dropdown-Menü finden Sie weitere Optionen (Zeile rechts oben). Abhängig vom Menü variiert die Auswahl. Hier sind möglichen Optionen:

  • Virtuelle Schalter: alle erstellten Virtuellen Schalter werden in einer Liste angezeigt.

  • Festwert: Sie können der Funktion einen festen Wert zwischen -100% und +100% zuweisen.

  • Funktionswert, wählen Sie eine Funktion aus der Liste.

  • Geber Zuweisung - Auswahl durch bewegen oder tippen.

Schalter Zuweisung - Auswahl durch schalten
Bewegen Sie den gewünschten Schalter. Dieser wird rot markiert und ein kleines Symbol erscheint. Das Symbol signalisiert den gewählten Schaltpunkt blau. Im Dropdown-Menü finden Sie weitere Optionen.

  • Virtuelle Schalter: alle erstellten Virtuellen Schalter werden in einer Liste angezeigt.

  • Fixierter Schalter, wähle Sie ob ein imaginärer Schalter immer offen oder geschlossen sein soll.

  • Schalter Zuweisung - Auswahl durch schalten

  • Flugphasenschalter, wählen Sie eine Flugphase aus der Liste, nun wird diese Flugphase wie ein virtueller Schalter agieren. Sie können die Logik mit Hilfe der "Inverse" Auswahlbox unten links ändern.
