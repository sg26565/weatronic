WiFi Setup - Hier sehen Sie die Konfiguration des Netzwerkes.

Es gibt mehrere Informationen.

  • Netzwerk-SSID, hier wird der Name des Netzwerks angezeigt.
  • BAT-Sender IP Adresse, falls Ihr Sender mit einem Netzwerk verbunden ist wird hier die zugeteilte IP Adresse angezeigt.
  • Sicherheit, bitte stellen Sie sicher das Ihr Netzwerk diese Verschlüsselung nutzt.
  • Passwort, Geben Sie bitte das entsprechende Passwort ein. Bitte achten Sie auf Groß- und Kleinschreibung.

    Durch Berührung des Kästchens "Passwort anzeigen" können Sie die Richtigkeit noch einmal überprüfen
