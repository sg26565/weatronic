Preflight Checklisten können individuell konfiguriert werden. Jede Liste kann mehrere Einträge beinhalten. Jeder Eintrag hat eine Überschrift (Titel) und ein frei zur gestaltendes Textfeld.
  • Klick auf "Plus", und ein neuer Punkt wird in die Checkliste aufgenommen.
  • Klick auf "Mülleiner", Einträge der Checkliste können gelöscht werden.
  • Klick auf „Neuanordnung", hier können Sie die Reihenfolge der Einträge in Ihrer Checkiste verändern.
Jede Liste hat eine Überschrift (Titel), ein Textfeld und eine Checkbox am Ende der Zeile.
  • Titel, geben Sie den Titel des Eintrages ein. Dieser Titel wird immer angezeigt.
  • Text, klicken Sie hier und Sie gelangen zu einem großen Feld, in dem Sie Ihren Text frei eingeben können. Diesen Text können Sie zusätzlich durch viele Optionen editieren, einfärben, vergrößern, etc. Klicken Sie auf die Fläche und es öffnet sich ein Texteditor.
  • Activ, markieren Sie hier den Eintrag. Nur markierte Einträge werden in den "Startup Warnings" angezeigt.
    		Anmerkung: Eine "PreFlight Checkliste" kann mehreren Modellspeichern zugeordnet werden. Aber die "Avtiv" Markierungen sind Modellspeicherunabhängig.
