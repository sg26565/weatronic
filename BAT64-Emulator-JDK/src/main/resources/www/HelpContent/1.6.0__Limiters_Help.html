Das Limiter Menü - wie immer benutzen Sie das "PLUS" Symbol um eine neuen Limiter hinzuzufügen. Oder das "Eimer" Symbol um einen Limiter zu löschen. Meist wird diese Option aus Sicherheitsgründen eingesetzt. Ein Beispiel ist es, dem Motor einen zusätzlichen Schalter zuzuordnen, damit dieser nicht versehentlich anlaufen kann. Das Limit kann für "beide" Seiten einer Funktion separat festgelegt werden. Die Liste zeigt alle erstellten Limiter und ebenso die Ziel Funktion jedes Limiters. Tippen Sie auf das Setup Feld betreten Sie das jeweilige Limiter Einstellungs Menü.

  • Limiter Name, benennen Sie den Limiter.

  • Funktion, hier wird der Name der Ziel Funktion angezeigt.

  • Limiter Setup, hier gelangen Sie in die jeweilige Limiter Konfiguration.
