WiFi Einstellung - Mit der WiFi Option haben Sie Möglichkeit, sehr elegant über andere WiFi-fähige Endgeräte Ihre Modelle zu konfigurieren. Grundsätzlich ist das WiFi bei jedem Einschaltvorgang deaktiviert.

  • Um die WiFi Funktionalität zu nutzen müssen Sie zuerst einmal Ihren BAT Sender mit einem WiFi Netzwerk verbinden.

    Aktivieren Sie zuerst das WiFi, in dem Sie das graue Kästchen links oben berühren ("WiFi aktiviert"). Bitte vergewissern Sie sich zuerst, dass ein WiFi-Stick in Ihrem Sender eingebaut wurde. Dieser muss am linken, internen USB Port des Senders stecken (von unten gesehen).
    Berühren Sie anschließend den Button "Scannen" und warten Sie einen Moment. Alle verfügbaren Netzwerke werden in der Liste angezeigt. Sind mehrere Netzwerke verfügbar, scrollen Sie nach unten, in dem Sie sanft mit Ihrem Finger über die Liste "wischen".
    Alle bereits gespeicherten Netzwerke sind ganz oben in der Liste zu finden. Auch wenn Sie ausserhalb der Reichweite sind.
    Das kleine WiFi Symbol zeigt Ihnen die Signalqualität des jeweiligen Netzwerkes an.
    Haben Sie das gewünschte Netzwerk gefunden, berühren Sie die Zeile. Sie wird nun hell hinterlegt. Berühren Sie nun den Button "Verbinden" und die Verbindung wird aufgebaut. Melden Sie sich das erste Mal in einem Netzwerk an, öffnet sich automatisch das WiFi-Setup.
    Nachdem das kleine Hinweisfenster für den Verbindungsaufbau verschwunden ist, erscheint eine Sender IP-Adresse am unteren linken Rand.
    Natürlich können Sie gespeicherte WiFi Netzwerke mit Hilfe des "Eimer" Symbol auch löschen. Tippen Sie auf das Setup um das Passwort für das Netzwerk einzustellen.

  • Jedes Gerät, was in Ihrem gewählten Netzwerk angemeldet ist, kann nun genutzt werden, um Eingaben für die Programmierung Ihres Senders vorzunehmen.

    öffnen Sie den Browser und geben Sie die IP-Adresse des Senders direkt in die Adresszeile ein. Nun bestätigen Sie die Eingabe und es erscheint 1:1 der Startbildschirm Ihres Senders. Von jetzt an werden sämtliche Eingaben, die Sie dann von anderen Geräten vornehmen, direkt im Sender vorgenommen und gespeichert.
    Bitte beachten Sie: Jedes Gerät des Netzwerkes kann den BAT Sender unabhängig und gleichzeitig programmieren.
    Sie können mehr als ein Gerät gleichzeitig benutzen.
ANMERKUNG: Das WiFi befindet sich zur Zeit in einer öffentlichen Beta-Phase. Auf Grund der Vielzahl unterschiedlicher Endgeräte kann es in Ausnahmefällen nicht funktionieren, was wir an dieser Stelle entschuldigen möchten. Wir werden dies so schnell wie möglich beheben.
Bitte beachten Sie auch die folgenden Hinweise für eine externe Programmierung des Senders:
  • Bei einem externen Zugriff wird die Tastatur des externen Gerätes selbst genutzt.
  • Vermeiden Sie die gleichzeitige Programmierung der selben Funktion oder des selben Menüs von unterschiedlichen Geräten.
  • Wir empfehlen Ihnen, das WiFi nach der externen Programmierung wieder zu deaktivieren. Bei der Datenübertragung könnte es zu einer leichten Beeinflussung des Rückkanals kommen.
