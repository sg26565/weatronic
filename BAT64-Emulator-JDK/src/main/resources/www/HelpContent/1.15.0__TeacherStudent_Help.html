Ab Software Version 6.07 ist ein Lehrer-Schüler Menü verfügbar. Hierdurch können Sie ein Modell mit 2 Sendern steuern.

  • Um die 2 Sender zu verbinden wird ein Standard 3.5mm Stereo-Klinkenkabel benötigt:
  • Der Pilot / Operator Mode kann nur in Verbindung mit zwei BAT 6x Sendern betrieben werden. (Digitales Protokoll)
  • Der Lehrer / Schüler Mode kann in Verbindung mit jedem Sender der ein PPM Signal liefert bzw. verarbeiten kann genutzte werden.
  • Der Schüler und der Operator Mode deaktivieren die 2,4GHz Funkstrecke.
  • Es stehen 6 unterschiedliche Modi zur Verfügung.
    • PPM AUS, dies ist die standard Einstellung. Die PPM Buchse gibt kein Signal aus.
    • PPM EIN, nun gibt die PPM Buchse ein Standard PPM Signal (12 Kanäle) aus, welches zum Beispiel für Flugsimulatoren benötigt wird.
    • Lehrer, dieser Mode gibt Ihnen die Möglichkeit einen beliebigen Sender mit einem PPM Signal als Schüler Sender zu benutzen.
      • Wählen Sie zunächst einmal einen Geber mit dem Sie den Schülersender freischalten. Falls kein gültiges PPM Signal erkannt wird sind die Lehrer Geber automatisch aktiv.
    • Schüler, der Sender wird die 2,4GHz Funkstrecke deaktivieren und ein Standard PPM Signal an der Klinkenbuchse ausgeben.
    • Pilot, dieser Modus ist nur in Verbindung mit einem zweiten BAT Sender mit „Operator“ Mode sinnvoll.
      • Wählen Sie zunächst einen Geber mit dem Sie den Operator Sender aktivieren bzw. deaktivieren können. Falls deaktiviert bzw. falls kein gültiges BAT Signal vorhanden ist, werden automatisch die „Backup Geber“ aktiv.
      • Nun können Sie einer Funktionen einen „zusätzlichen Geber“ zuweisen. Sie finden die 12 Geber im allgemeinen Geberzuweisungsmenü als Auswahl.
    • Operator, dieser Modus ist nur in Verbindung mit einem zweiten BAT Sender mit „Pilot“ Mode sinnvoll.
