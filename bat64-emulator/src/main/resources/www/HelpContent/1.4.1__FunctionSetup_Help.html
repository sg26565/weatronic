Funktions Einstellung - konfigurieren Sie die Funktion nach Ihren Bedürfnissen. ANMERKUNG: Die Kurve bezieht sich nur auf den Geber der Funktion.
INFO: Der Begriff "Dual-Rate" wird durch den Begriff "Flex Rate" ersetzt. Da der eingestellte Wert mit einem Geber stufenlos (flexibel) im Flug angepasst werden kann. Das gleiche gilt für die Exponential- und Differential Option.

ANMERKUNG: Alle unterstrichen Werte sind Flugphasenabhängig.

  • Flex Rate, hier können Sie einen Wert zwischen 0% und 100% einstellen. Ebenso können Sie einen Geber zuweisen um den Wert zu ändern.

  • Flex Expo, hier können Sie den Funktions-Geber mit einer Exponential Kurve modifizieren. Sie können dem Flex-Expo Wert einen Geber zuweisen und diesen somit stufenlos im Flug anpassen.

  • Flex Differenzierung, hier können Sie für die Funktion einen so genannten "Differenzierung" Wert wählen. Diese Option setzt voraus das die zugeordneten Servos der linken(L) oder rechten(R) Seite des Tragflügels oder des Leitwerkes zugeordnet sind.
    		ANMERKUNG: Im Allgemeinen wird die Differenzierung bei der Querruder Funktion benötigt. Der Wert wird normalerweise nur als positiver Wert verwendet, sprich die Querruder bewegen sich weniger weit nach Unten als nach Oben. Dies soll dem negativen Wendemoment entgegen wirken.

  • Als Fail Safe Position wird die Position der Funktion bezeichnet, die eingenommen wird, wenn der Empfänger kein gültiges Signal vom Sender empfängt und die Verbindung zwischen Sender und Empfänger unterbrochen wird.
    Wählen Sie zwischen einem festen Wert oder einer "Halten" Funktion. Wird "Halten" gewählt, bleibt die Funktion in der dann aktuellen und zuletzt übertragenen Position. Mit der Auswahl "Wert" kann eine individuelle Position zugeordnet werden, was wir für jede Funktion empfehlen. In der Kurven-Grafik wird die Fail Safe Position mit einer roten Linie dargestellt. ACHTUNG: Kontrollieren Sie Ihre Fail Safe Einstellungen vor jedem Betrieb ihres Modells! Ebenfalls können Sie in den "Allgemeine Einstellungen" die eingestellten Fail Safe Positionen im Menü "Sicherheitstests" kontrollieren.

  • Funktionen können mit einer Verzögerung hinterlegt werden, mit der sie ausgeführt werden. So kann zum Beispiel dem Ausklappen eines Fahrwerkes eine Zeit von bis zu 20 Sekunden hinterlegt werden. Das Fahrwerk wird dann für die Ausführung der gesamten Funktion genau diese Zeit brauchen. Der Umkehrfunktion kann eine separate Zeit hinterlegt werden, ebenfalls maximal bis zu 20 Sekunden. Die Funktion kann für beide Richtungen in Schritten von 0,1 Sekunde verlangsamt werden.

  • Tippen Sie auf die Schaltfläche mit dem "Stift" Symbol um den Kurven Editor zu öffnen.
