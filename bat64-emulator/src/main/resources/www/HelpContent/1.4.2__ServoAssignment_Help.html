Servo Zuordnung - hier werden Servos bzw. Servosteckplätze mit einer Funktion verknüpft.
Tippen Sie auf das "PLUS" Symbol und wählen Sie im Pop-Up das Servo bzw. den Steckplatz. Mit dem "Eimer" Symbol können Sie zugeordnete Servos wieder löschen. Falls die Funktion mit dem "Setup Wizard" erstellt wurde können Sie die Zuordnung nur über die Servo Konfigurations Grafik ändern.
Die Servoliste hat mehrere Spalten:

  • Pos. (Servosteckplatz Nummer), hier wird die Position des Servosteckplatzes angezeigt.

  • Servo, hier sehen Sie die Bezeichnung für den Steckplatz. Im Servo Konfiguration Menü können Sie diesen ändern.

  • Gruppe, hier wird evtl. der Gruppenindex des Servos angezeigt.

  • L/R, hier kann man die Lage des Servos im Tragflügel bzw. im Leitwerk hinterlegen. L steht für linke Seite und R für rechte Seite (in Flugrichtung gesehen). Diese Information wird hauptsächlich für die Querruder Differenzierung verwendet.
    		ANMERKUNG: Sie können dieses Attribut nicht ändern, falls die Funktion mit Hilfe des "Setup Wizard" erstellt wurde.

  • - Weg +, hier können Sie den Weg für die beiden Seiten jedes Servos für jede Funktion separat einzeln einstellen. Haben Sie einen Geber für diese Funktion zugeordnet wird die aktuelle Seite blau markiert. Nutzen Sie den Tausch-Button um die Laufrichtung des jeweiligen Servos durch tauschen der Vorzeichen und Werte umzudrehen.
    		ANMERKUNG: Wenn Sie eine Kurve einstellen wird der "normale" plus minus Weg durch eine "Curve Reset" Schaltfläche ersetzt. Wenn Sie auf die Schaltfläche klicken wird die eingestellte Kurve zurückgesetzt und der plus minus Weg wieder dargestellt.

  • Kurven Editor, tippen Sie auf den Kurven Editor Button (kleines "Stift" Symbol) um die Kurve mit bis zu 33 Punkten anzupassen
