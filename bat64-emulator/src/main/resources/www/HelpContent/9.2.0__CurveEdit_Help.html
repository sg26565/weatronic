Kurven Editor
Hier können Sie die Kurve mit bis zu 33 Punkten anpassen. Alle Auswahlpunkte werden als kleine graue Punkte auf der Kurve dargestellt. Schwarze Punkte symbolisieren Punkte auf der Kurve, die fixiert wurden. Sollten Sie diese Fixierung aufheben wollen, können Sie diese Punkte wieder markieren und bewegen. Ausgewählte Punkte werden rot dargestellt und können von Ihnen bewegt werden.
Links sehen Sie die visualisierte Kurve im Koordinatensystem. Der Geber wird auf der horizontalen Achse von links -100% nach rechts +100% dargestellt, und die Function wird auf der vertikalen Achse von Unten -100% nach Oben +100% dargestellt.
Rechts neben dem Koordinatensystem finden sie mehrere Schaltflächen.

  • Die komplette Kurve kann vertikal oder horizontal gespiegelt werden.

  • Wählen Sie mit dem Button "Vorlage" eine vordefinierte Kurve aus. Auch diese können Sie bei Bedarf weiter bearbeiten.

  • Sie haben die Wahl zwischen einer allgemein gültigen Kurve oder einer Flugphasen abhängigen Kurve. Bei dem gewählten Status der Kurve wird das Symbol weiß hinterlegt.
    • "allgemein Gültig (Global)": die Kurve ist für jede Flugphase gleich – dies ist die "Standard"-Einstellung.
    • "Flugphasen abhängig (nicht-Global)": die Kurve ist für jede Flugphase individuell anpassbar

  • Auswählen einzelner Punkte (nur bei grauen Punkten möglich). Bewegen Sie den Cursor mittels des Gebers an den Punkt auf der Kurve, den Sie auswählen möchten. Nun berühren Sie den Button "Hinzufügen". Der gewählte Punkt wird nun rot markiert.

  • Punkte bewegen (alle roten Punkte). Sie können alle rot markierten Punkte bewegen. Für eine schnelle Einstellung berühren Sie unabhängig von der Position des Gebers einfach die Stelle im Koordinatensystem, zu der sich der Punkt bewegen soll. Die Feinabstimmung erfolgt mittels der "+" und "-" Tasten. Es besteht ebenfalls die Möglichkeit, mit einem der 6 digitalen Endlos-Drehgeber die Position des roten Punktes zu verändern.
    		ANMERKUNG:Bitte beachten Sie, dass bei der Nutzung der Endlos Drehgeber sich evtl. Werte ändern, sollte der gewählte Geber bereits als Trimmer einer Funktion zugeordnet sein.

  • Punkte Auswählen (nur bei schwarzen Punkten möglich):
    • Bewegen Sie den Cursor mittels des Gebers auf den schwarzen Punkt, den Sie auswählen möchten. Berühren Sie den "Auswahl" Button und der gewählte Punkt wird rot markiert.
    • Wenn keine roten Punkte auf der Kurve vorhanden sind, können Sie mit dem Button "Alle" alle schwarzen Punkte wählen. Diese werden dann rot und Sie können wie beschrieben die gesamte Kurve bewegen.
    • Sie können auch direkt auf dem Touchscreen einen schwarzen Punkt berühren. Dieser wird dann ausgewählt und wird rot.

  • Punkte Abwählen (nur bei roten Punkten möglich):
    • Bewegen Sie den Geber über den ausgewählten roten Punkt. In diesem Moment wird der Button "Abwählen" frei gegeben und Sie können ihn berühren. Der rote Punkt wird schwarz.
    • Alternativ können Sie auch den Button "Keiner" berühren, mit dem alle roten Punkte auf der Kurve wieder schwarz werden.

  • Punkte Löschen (nur bei roten Punkten möglich): Bewegen Sie den Geber über den ausgewählten roten Punkt. Es erscheint ein "Löschen" Button. Berühren Sie diesen Button und der Punkt wird wieder grau.

Jedes Mal, wenn eine Kurve verändert wurde, müssen die Änderungen mit dem Haken bestätigt werden. Erst dann werden sie gespeichert. Wollen Sie die Änderungen verwerfen, drücken Sie das Kreuz ("X"). Danach können Sie mit dem "Seite Verlassen" Symbol die Seite schließen. Möchten Sie gleichzeitig mehrere Punkte der Kurve bewegen, markieren Sie diese erst alle und verändern dann die Position.

Kurveneditor - Dieser spezielle Kurveneditor behandelt die Abweichungskurve der "Slave" Servos in Relation zur Bewegungskurve der "Master" Servo.
Der Kurveneditor erlaubt Ihnen dem "Slave-servo" eine Abweichungskurve mit mehreren Punkten (Anzahl der Punkte wird durch die Genauigkeit vorgegeben) einzustellen, links sehen Sie die grafische Darstellung der Abweichung (plus/minus) als Kurve.
Rechts sehen Sie einige Schaltflächen um die Kurve einzustellen. Auch werden Servoposition und falls ein Gizmo vorhanden ist der RX-Strom angezeigt:
  • Mit den "PLUS" und "MINUS" Buttons bewegen Sie den rot markierten Punkt auf der Kurve. Ebenso können Sie entweder mit dem zugeordneten Endlosdrehgeber oder aber auch per Toucheingabe den Punkt verschieben.
    		ANMERKUNG:Bitte beachten Sie, dass bei der Nutzung der Endlos Drehgeber sich evtl. Werte ändern, sollte der gewählte Geber bereits als Trimmer einer Funktion zugeordnet sein.
  • Mit dem "VORHERIGER" und "NÄCHSTER" Button können Sie den Punkt auswählen den Sie bewegen möchten. Er ist dann rot markiert. Die Servos der Gruppe bewegen Sich dann auch automatisch zu der Position des Punktes.
  • Mit dem "ALLE" Button markieren Sie alle Punkte der Kurve und somit bewegen Sie die komplette Kurve.

Kurvenänderungen können Sie mit Hilfe des "KREUZ" Buttons verwerfen, genauso müssen Sie jede Änderung mit dem "HAKEN" Button zuerst bestätigen bevor Sie das Menü verlassen können.
