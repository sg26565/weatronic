Setup Ansicht - Hier wird Ihnen die Funktionsliste als reine Setupliste dargestellt. Sie sehen hier alle flugphasenabhängigen Werte aus dem Funktions setup auf einen Blick.

  • Setup: Verknüpfung zum eigentlichen Setup der Funktion

  • Flex Rate, hier können Sie einen Wert zwischen 0% und 100% einstellen.

  • Flex Expo, hier können Sie den Funktions-Geber mit einer Exponential Kurve modifizieren.

  • Flex Differenzierung, hier können Sie für die Funktion einen so genannten "Differenzierung" Wert wählen. Diese Option setzt voraus das die zugeordneten Servos der linken(L) oder rechten(R) Seite des Tragflügels oder des Leitwerkes zugeordnet sind.
    		ANMERKUNG: Im Allgemeinen wird die Differenzierung bei der Querruder Funktion benötigt. Der Wert wird normalerweise nur als positiver Wert verwendet, sprich die Querruder bewegen sich weniger weit nach Unten als nach Oben. Dies soll dem negativen Wendemoment entgegen wirken.

  • FPO, der "Flugphasen-Offset" gibt Ihnen die Möglichkeit verschiedene "Offsets"(Abweichungen) für jede Funktion zu vergeben. Die obere "Informations Zeile" zeigt Ihnen die aktuellen Flugphase.
    		ANMERKUNG: Die kleine "dreifach-status" Grafik wird als Information zusätzlich angezeigt. Somit können Sie die Stellung eines evt. zugordneten Geber kontollieren.
