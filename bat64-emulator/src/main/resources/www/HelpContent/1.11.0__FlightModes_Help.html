Flugphasen sind sehr nützlich, wenn die allgemeinen Einstellungen Ihres Modells während des Fluges bestimmten Situationen angepasst werden sollen.
Sie können durch berühren der "PLUS" Taste Flugphasen hinzufügen. Diese werden am Ende der List hinzugefügt. Bitte beachten Sie das die Werte der Flugphase "NORMAL" für die neue Flugphase übernommen werden. Berühren Sie das "Eimer" Symbol um Flugphasen zu löschen.
Zusätzlich sehen Sie einen "Kopieren" Button. Hier können Sie eine Flugphase kopieren. Alle Flugphasenabhängigen Werte (die unterstrichenen Werte) werden für die Kopie übernommen.
Es gibt viele Werte die Flugphasen
"abhängig -> nicht-global" oder
"unabhängig -> global" sind,
meistens kann man wählen. ANMERKUNG: Flugphasen-abhängige Werte sind unterstrichen dargestellt.
Berühren Sie "Schalterzuweisung", dieses Menü erlaubt Ihnen Schalter (auch Virtuelle Schalter) den Flugphasen zuzuordnen und zwischen diesen umzuschalten.

  • In der Spalte "Name" können Sie jeder Flugphase einen bestimmten Namen geben. Berühren Sie die Schaltfläche, um diesen zu ändern.

  • Für einen sanften Wechsel von einer Flugphase in eine andere können Sie die Übergangszeit ("Zeit") selber definieren. Diese Zeit ist in Schritten von 0,1 Sekunden einstellbar.

  • Wollen Sie besondere Funktionen innerhalb der Flugphasen ohne Zeitverzögerung umschalten (Funktionen ohne Überblendzeit), sind diese in der entsprechenden Zeile der Flugphase einzutragen.
