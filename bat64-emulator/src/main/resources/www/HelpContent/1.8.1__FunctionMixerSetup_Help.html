Mischer Einstellung - stellen Sie hier Ihren Mischer ein

  • "Von" Funktion, ist die Funktion die als Eingang dient

  • "Trimm", hier können Sie wählen ob der Trimm der "Von" Funktion mit einbezogen werden soll

  • "Auf" Funktion, die Funktion die gemischt werden soll

  • Vererben, wird diese Option aktiviert, wird der auf die "Auf" Funktion zugemischte Wert bei allen anderen Mischern berücksichtigt bei der diese Funktion eine "Von" Funktion ist. Standardmäßig werden Mischanteil nicht vererbt.
    		ANMERKUNG: Bitte benutzen Sie diese Option mit Bedacht, da hierdurch sehr schnell gegenseitige zirkulare Abhängigkeiten und entsprechend unerwünschte Fehlfunktionen entstehen können.

  • Zeitverzögerung kann in Schritten von 0,1 Sekunden für beide Richtungen separat eingestellt werden
    		ANMERKUNG: Die Zeitverzögerung ist Flugphasen abhängig.

  • Flex Anteil, legen Sie den Mischer auf einen Geber oder tragen Sie einen festen Wert ein.
    		ANMERKUNG: Diser Wert ist Flugphasen abhängig.

  • Tippen Sie auf das "Stift" Symbol und erstellen Sie eine individuelle Kurve. Der "Flex Anteil" Wert wirkt hierbei als Stellfaktor für die Kurve.
